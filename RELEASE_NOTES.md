## kkImgDesktop 2.0.0

このリリースは, 従来の Python/Flet 版からのメジャーアップデート (フルスクラッチ書き換え) となります.
現在, UIの構築を中心に進めており, 過去の機能はまだほとんど実装されていません.

### 今回の変更点

- React Native for macOS への移行
  - アプリの基盤を React Native for macOS に変更しました.
  - 可能な限り macOS ネイティブのコンポーネント, レイアウト, 操作体系 (ドラッグ&ドロップ, 複数選択など) を意識して作っています.
  - 一部リスト表示などでは総合的な判断でRNのコンポーネントを使用しています.
  
- 今後の予定
  - 現在は macOS 向けですが, Windows 版に関しても追って対応予定です.
  - 旧バージョンに存在した, ファイル名を元にしたEXIF日時の推測と書き込み, AVIF変換機能などのコアロジックは順次移植していきます.

今回は新しいUIができたという初期バージョンとしてのリリースとなります.
